# SovereignLayer Identity Engine

## Overview
SovereignLayer is a Web3 identity intelligence engine that analyzes blockchain wallets and creates decentralized reputation profiles.

- BaseIndexer
- TransactionScanner
- BaseLogScanner
- WalletAge

- GraphBuilder
- SybilDetector
- ReputationEngine
